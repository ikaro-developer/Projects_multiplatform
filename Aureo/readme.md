# Sobre o Projeto

<p align="center">
<strong>Aureo</strong> é uma aplicação desktop (com suporte futuro para web) desenvolvida com foco em **gestão financeira pessoal.  
O sistema facilita o controle de transações, categorização de despesas e receitas, além de oferecer uma visualização clara e organizada do fluxo financeiro do usuário.
Com o Aureo, é possível acompanhar entradas e saídas, visualizar detalhes de cada transação e entender melhor onde e como o dinheiro está sendo utilizado.

</p>
